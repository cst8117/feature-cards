# Style Guide

## Layout
The designs represent the following layout breakpoints:

- Mobile: < 768px
- Tablet: 768px <=> 1028px
- Desktop: > 1028px

## Colors
- Light Gray: #f7f7f7 
- Dark Gray: #484848
- Cyan: #45d3d3
- Red: #ea5353
- Orange: #fcaf4a
- Blue: #549ef2
